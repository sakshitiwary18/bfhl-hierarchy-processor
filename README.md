# BFHL Hierarchy Processor

This project implements the SRM Full Stack Engineering Challenge.

## Features
- REST API: POST /bfhl
- Validates node relationships
- Handles duplicates and invalid inputs
- Builds hierarchical trees
- Detects cycles
- Computes summary metrics

## Tech Stack
- Node.js (Express)
- HTML, CSS, JavaScript

## Run Locally
cd bfhl-api
npm install
node index.js

Open index.html in browser.

## API Endpoint
POST /bfhl
